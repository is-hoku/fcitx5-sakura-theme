## Installation

Clone this repository.

```
git clone https://github.com/sakura-theme/fcitx5.git ~/.local/share/fcitx5/themes/sakura
```

Add the following line to `~/.config/fcitx5/conf/classicui.conf`.

```
Theme=sakura
```
